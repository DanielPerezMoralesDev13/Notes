# Lista con viñetas

**Listas**: *Puedes crear una lista sin orden (lista con viñetas) utilizando asteriscos, guiones o más signos. Por ejemplo:*

```markdown
* Elemento 1
* Elemento 2
* Elemento 3

* Elemento
  * SubElemento
```

*Se renderizará como:*

* Elemento 1
* Elemento 2
* Elemento 3

* Elemento 1
  * SubElemento
